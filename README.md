# 📘 Calculadora Pedagógica GUI — v2.0.0

Aplicação em Python desenvolvida para **automatizar o cálculo de notas bimestrais**, reduzindo o trabalho manual do professor e oferecendo uma interface gráfica simples, direta e acessível.  
A ferramenta foi projetada para uso rápido em ambiente escolar, com foco em **produtividade**, **clareza** e **mínima curva de aprendizado**.

---

## 🎯 Objetivo do Projeto

Professores frequentemente precisam calcular médias, pesos e fechamentos de notas repetidas vezes. Este projeto busca:

- eliminar cálculos manuais repetitivos
- reduzir erros humanos
- agilizar o fechamento de notas
- oferecer uma interface visual intuitiva
- permitir uso imediato sem instalação complexa

---

## 🏛️ Arquitetura e Decisões Técnicas

A aplicação foi construída com foco em **simplicidade**, **portabilidade** e **manutenibilidade**:

- **Isolamento de ambiente (`venv`)** — garante integridade das dependências e reprodutibilidade.
- **Tkinter (GUI nativa do Python)** — evita dependências externas e permite execução em qualquer máquina com Python instalado.
- **Separação entre lógica e interface** — facilita manutenção e evolução futura.
- **Interface minimalista** — reduz ruído visual e prioriza fluxo rápido de entrada e saída.

---

## 🧪 Tecnologias Utilizadas

- **Python 3.13**
- **Tkinter** (interface gráfica nativa do Python)
